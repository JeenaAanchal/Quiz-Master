# Quiz Master

A responsive quiz application built with Django and HTML Templates, offering category-based quizzes, dynamic question rendering, and instant score evaluation. Designed with a modular backend for easy upgrades and clean session handling.

Features
Responsive UI built with HTML, CSS, and Django templates
Category-based quiz selection
Dynamic question loading and real-time score calculation
Django session management for user progress
Easily extendable backend structure

Tech Stack
Backend: Django
Frontend: HTML, CSS (Responsive Layout), JavaScript
Database: Postgresql

How to Run

Clone the repo
git clone <your-repo-url>
cd quiz-master

Install dependencies
pip install -r requirements.txt

Apply migrations
python manage.py migrate

Run the server
python manage.py runserver


Open in browser:
http://127.0.0.1:8000/
